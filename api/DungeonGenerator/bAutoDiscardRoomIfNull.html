Auto Discard Room if Null

Class: Dungeon Generator
Category: Procedural Generation
Type: Boolean
Editor Access: Anywhere
Blueprint Access: Read/Write

Description​

If true, returning null in ChooseNextRoom is the same as calling DiscardRoom.
